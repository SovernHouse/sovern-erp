ARCHITECTURE NOTES
================================================================================

WORKSPACE STRUCTURE:
- Root package.json defines workspaces
- Shared package is npm workspace dependency
- Automatic symlink via npm workspaces
- No need for separate build step

COMPONENT PHILOSOPHY:
- Stateless where possible
- Props-driven configuration
- Tailwind CSS for styling
- Lucide React for icons
- DayJS for date handling

API INTEGRATION:
- Centralized axios client
- Auth token in localStorage
- 401 auto-redirect to login
- Interceptor-based error handling

COLOR SCHEME:
- Primary: blue-600 (blue-700 on hover)
- Status colors: Tailwind palette
- Error: red-600
- Success: green-600
- Warning: yellow-600
