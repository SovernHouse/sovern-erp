================================================================================
TROUBLESHOOTING
================================================================================

Problem: Tests timeout waiting for servers
Solution: Increase timeout in playwright.config.js or ensure ports are free

Problem: Selectors not found
Solution: Run in debug mode (npx playwright test --debug) to inspect

Problem: Port already in use
Solution: Kill the process (lsof -ti:5000 | xargs kill -9)

Problem: Login always fails
Solution: Verify credentials match actual setup, check selector patterns

Problem: Browser not found
Solution: May need to install: npm install @playwright/test && npx playwright install chromium

EXTENDING THE TESTS
================================================================================

To add new tests:

1. Create test file in appropriate directory:
   /e2e/admin-portal/my-feature.spec.js

2. Write test using Playwright syntax:
   const { test, expect } = require('@playwright/test');
   
   test.describe('My Feature', () => {
     test('does something', async ({ page }) => {
       await page.goto('/');
       // Test code
     });
   });

3. Run the test:
   npx playwright test e2e/admin-portal/my-feature.spec.js
